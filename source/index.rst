Welcome to Dripline!
====================

Dripline provides a framework for building hardware monitoring and controls systems, targeted at medium-scale physics experiments.
This is just the homepage, you can find more information at the following pages:

* `The Dripline Protocol Documentation <driplineorg.github.io/dripline>`_
* `The Guides and Walthroughs <driplineorg.github.io/controls-guide>`_
